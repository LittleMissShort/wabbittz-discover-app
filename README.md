# Wabbittz Discovery 🎨

## Overview
Wabbittz Discovery is a web application that allows users to explore and discover random artwork from the Harvard Art Museums API. The app intelligently filters results and displays only valid artwork with images and key details for an engaging user experience.

## Features
- Discover random artwork
- Display artwork images, artist, culture, and date
- Smart filtering system (ban list)
- Dynamic and interactive interface

## Technologies Used
- React
- JavaScript
- Harvard Art Museums API
- CSS
- Vite
